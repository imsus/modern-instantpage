# instant.page

**Make your site's pages instant in 1 minute and improve your conversion rate by 1%.**

:information_source: Info is on [the website](https://instant.page).

## Install

```bash
npm install instant.page
```

## Usage

### Script tag

```html
<script src="dist/instantpage.min.js"></script>
```

### ES Module

```javascript
import 'instant.page'
```

## Development

### Prerequisites

- Node.js 18+
- npm

### Setup

```bash
npm install
```

### Commands

```bash
# Build the library
npm run build

# Run tests
npm test

# Run tests in watch mode
npm run test:watch

# Type check
npm run typecheck

# Start dev server
npm run dev
```
